# Task 2 - Data Pipeline Building

This project demonstrates a simple data engineering pipeline using the CoinGecko public API.

## Pipeline Steps
1. Fetch cryptocurrency market data from CoinGecko API
2. Transform and clean raw JSON data using pandas
3. Create derived analytical fields
4. Prepare transformed data for BigQuery storage

## Technologies Used
- Python
- Google Colab
- Pandas
- CoinGecko API
- BigQuery
